# Meta Images

This directory contains images used for SEO and social media sharing:

- `og-image.jpg` - Open Graph image (1200x630px) for social media sharing
- `apple-touch-icon.png` - iOS touch icon (180x180px)
- `icon-192.png` - PWA icon (192x192px)
- `icon-512.png` - PWA icon (512x512px)

## Creating Images

You can create these images using any design tool. Recommended specifications:

1. **og-image.jpg** (1200x630px)
   - Use for Facebook, LinkedIn, Twitter cards
   - Include: Site name, tagline, visual branding
   - Format: JPG/PNG

2. **apple-touch-icon.png** (180x180px)
   - iOS home screen icon
   - Format: PNG with transparency

3. **icon-192.png** & **icon-512.png**
   - PWA icons for Android
   - Format: PNG with transparency
